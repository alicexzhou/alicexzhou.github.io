Patient Data Exploration Dashboard

Project Description

I developed a web application using Dash by Plotly (Python) to help data scientists explore ICU patient data. Each dataset comprised of thousands of ICU patients and hundreds of variables for each patient, starting from admission to the ICU to dischange. Users were able to dynamically graph and highlight their variables of choice. This application was used and repurposed for countless projects.
